## 修改内容

### 1. 删除标题栏左侧标题 (TaskList.tsx)

**修改位置**: `src/webview/components/TaskList.tsx` 第 1090-1092 行

**修改前**:
```tsx
return React.createElement('div', { className: 'app' },
  React.createElement('header', { className: 'header' },
    React.createElement('h1', null, displayTitle),
    React.createElement('div', { className: 'header-actions' },
```

**修改后**:
```tsx
return React.createElement('div', { className: 'app' },
  React.createElement('header', { className: 'header' },
    React.createElement('div', { className: 'header-actions' },
```

移除了 `React.createElement('h1', null, displayTitle)` 这一行，不再显示左侧标题。

### 2. 优化 header-actions 布局样式 (bundle.css)

**修改位置**: `src/webview/bundle.css` 第 24-30 行

**修改前**:
```css
.header-actions {
  display: flex;
  gap: 8px;
  align-items: center;
}
```

**修改后**:
```css
.header-actions {
  display: flex;
  gap: 4px;
  align-items: center;
  flex-wrap: nowrap;
  overflow: hidden;
}
```

**修改说明**:
- 将 `gap` 从 8px 减小到 4px，使元素更紧凑
- 添加 `flex-wrap: nowrap` 防止元素换行
- 添加 `overflow: hidden` 确保超出部分隐藏而不是推挤布局
