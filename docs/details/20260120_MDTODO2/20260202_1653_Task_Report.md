# R54.9 任务报告

## 任务描述

增加一个功能，执行按钮可以选择使用Claude/OpenCode，这个功能通过webview的顶部状态栏右侧加一个配置按钮，点击配置按钮后弹出一个配置面板，在配置面板中配置，配置后的结果记录到.vscode/mdtodo/settings.json中持久化。

**状态**：[completed]

在 Toolbar.tsx 中添加配置按钮（齿轮图标）：
- 新增 `handleOpenSettings` 回调函数类型
- 使用现有 `toolbar-icon-btn` 样式类
- 按钮放置在 toolbar 最右侧

### R54.9.2 创建配置面板组件

**状态**：[completed]

创建 SettingsPanel 组件：

- 使用 `position: fixed`, `zIndex: 10000` 样式
- 在 TaskList 中集成，使用 state 控制显示/隐藏
- 变更时发送 `executionModeChanged` 消息到扩展端

### R54.9.5 添加配置相关的消息处理

**状态**：[completed]

修改 webviewProvider.ts：

### 功能说明

1. **配置入口**：Toolbar 右侧齿轮图标
2. **配置面板**：浮动的设置对话框，支持 Claude/OpenCode 选择
3. **持久化**：`.vscode/mdtodo/settings.json`
4. **执行路由**：根据配置选择 `claude` 或 `opencode` 命令执行
