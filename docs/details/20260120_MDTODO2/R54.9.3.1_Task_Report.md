## 实现内容

### 1. 新建 src/services/settingsService.ts

创建了设置服务类 `SettingsService`，提供以下功能：

- **读取设置**: `readSettings(workspacePath)` - 读取 `.vscode/mdtodo/settings.json` 文件
- **写入设置**: `writeSettings(workspacePath, config)` - 写入配置到 settings.json
- **更新执行模式**: `updateExecutionMode(workspacePath, mode)` - 更新 executionMode 配置项
- **获取执行模式**: `getExecutionMode(workspacePath)` - 获取当前 executionMode
- **确保目录存在**: `ensureSettingsDirectory(workspacePath)` - 创建 `.vscode/mdtodo/` 目录

#### 配置接口

```typescript
export interface SettingsConfig {
```
