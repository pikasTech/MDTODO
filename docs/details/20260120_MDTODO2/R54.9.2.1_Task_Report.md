# R54.9.2.1 SettingsPanel 组件 - 任务报告

## 任务概述

创建 SettingsPanel 组件，支持 Claude/OpenCode 执行模式选择。

## 需求分析

1. 创建 SettingsPanel 组件文件
2. 支持 Claude 和 OpenCode 两种执行模式选择（单选按钮）
3. 使用 props 控制显示/隐藏（isOpen, onClose）
4. 选择变更时调用 onChange 回调
5. 样式参考 ContextMenu（zIndex: 10000，浮动在最上层）
6. 在 TaskList 中集成 SettingsPanel，使用 state 控制显示/隐藏

## 实现详情

### 1. 新建文件

#### src/webview/components/Settings/SettingsPanel.tsx

组件功能：
- 支持 `claude` 和 `opencode` 两种执行模式
- 使用单选按钮进行选择
- 居中显示的对话框样式
- 点击外部或滚动时自动关闭
- 包含模式说明文字

Props 接口：
```typescript
interface SettingsPanelProps {
  isOpen: boolean;                    // 是否显示面板
  executionMode: ExecutionMode;       // 当前执行模式
  onClose: () => void;                // 关闭面板回调
  onChange: (mode: ExecutionMode) => void;  // 模式变更回调
}
```

样式特性：
- 定位：`position: fixed`, `zIndex: 10000`
- 居中：`top: 50%, left: 50%, transform: translate(-50%, -50%)`
- 背景：`#2d2d2d`，圆角 `8px`
- 阴影：`box-shadow: 0 8px 24px rgba(0, 0, 0, 0.5)`
- 选中状态高亮：`rgba(0, 122, 204, 0.15)` 背景色

### 2. 修改文件

#### src/webview/components/TaskList/hooks/useTaskListState.ts

新增状态：
- `settingsPanelOpen: boolean` - 设置面板是否打开
- `executionMode: 'claude' | 'opencode'` - 当前执行模式

新增状态更新函数：
- `setSettingsPanelOpen`
- `setExecutionMode`

#### src/webview/components/TaskList.tsx

1. 导入 SettingsPanel：
   ```typescript
   import { SettingsPanel, ExecutionMode } from './Settings';
   ```

2. 解构新增状态：
   ```typescript
   settingsPanelOpen,
   setSettingsPanelOpen,
   executionMode,
   setExecutionMode,
   ```

3. 新增处理器：
   - `handleOpenSettings` - 打开设置面板
   - `handleExecutionModeChange` - 处理执行模式变更，变更后发送消息到扩展端

4. Toolbar 增加 `handleOpenSettings` 属性

5. 在 TaskList 底部渲染 SettingsPanel：
   ```typescript
   React.createElement(SettingsPanel, {
     isOpen: settingsPanelOpen,
     executionMode,
     onClose: () => setSettingsPanelOpen(false),
     onChange: handleExecutionModeChange,
   })
   ```

### 3. 样式一致性

- 继承项目暗色主题配色（`#2d2d2d`, `#3c3c3c`, `#d4d4d4`）
- zIndex: 10000 与 ContextMenu 一致，确保在最上层显示
- 关闭按钮、选项高亮等交互反馈与现有组件风格一致

## 测试要点

1. 点击工具栏齿轮按钮可打开设置面板
2. 可以选择 Claude 或 OpenCode 模式
3. 选择变更后触发 onChange 回调，面板关闭
4. 点击面板外部可关闭面板
5. 面板居中显示在最上层
6. 选中项有视觉高亮反馈
