# R55.2 任务报告

## 任务描述
install.bat在打包的时候，应当排除 doc/ 目录

## 修改内容

修改了 `install.bat` 第23行的打包命令，添加了 `--ignore doc/` 参数：

**修改前：**
```bat
call npx vsce package --allow-missing-repository
```

**修改后：**
```bat
call npx vsce package --allow-missing-repository --ignore doc/
```

## 验证方法
执行 `install.bat` 进行打包，然后检查生成的 vsix 文件是否包含 doc/ 目录（应被排除）。
