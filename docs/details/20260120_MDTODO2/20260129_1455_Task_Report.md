# R55.9 任务报告

## 任务描述

修复 VSIX 打包时的仓库 URL 检测错误：
```
[3/5] Packaging VSIX...
 ERROR  Couldn't detect the repository where this extension is published.
 The image 'image.png' will be broken in README.md.
ERROR: Packaging failed!
```

## 问题原因

`package.json` 中缺少 `repository` 字段，vsce 打包器无法自动检测仓库 URL，导致 README.md 中的图片路径无法正确解析。

## 解决方案

在 `package.json` 中添加 `repository` 字段：

```json
"repository": {
  "type": "git",
  "url": "https://github.com/mdtodo/vscode-mdtodo"
}
```

## 修改文件

- `package.json:6-9` - 添加 repository 配置

## 验证结果

打包成功，输出：
```
DONE  Packaged: D:\Work\vscode-mdtodo\vscode-mdtodo-0.0.2.vsix (377 files, 1.79 MB)
```

打包产物包含 `image.png`，图片路径正确解析。
