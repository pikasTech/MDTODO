# R53.8 任务报告

## 任务描述

将是否完成的 checkbox 放置到标题栏的左边，仅次于 task id，这样不管其他图标是否显示，是否完成的 checkbox 位置不会变动。

## 修改内容

### 文件：`src/webview/components/TaskItem.tsx`

**修改前**：
```
task-id-wrapper
├── task-id
├── processing-badge (if processing)
├── link-status-icon (if has links)
├── task-checkbox  <-- 原来在后面
└── task-actions-inline
```

**修改后**：
```
task-id-wrapper
├── task-id
├── task-checkbox  <-- 移动到紧跟在 task-id 后面
├── processing-badge (if processing)
├── link-status-icon (if has links)
└── task-actions-inline
```

### 具体变更

将 checkbox 元素从 `link-status-icon` 之后移动到 `task-id` 之后、 `processing-badge` 之前。

## 验证结果

- 编译成功（`npm run compile`）
- webpack 打包成功
- 修改符合预期：checkbox 现在紧跟在 task id 后面，位置固定不变
