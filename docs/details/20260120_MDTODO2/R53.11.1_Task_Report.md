# R53.11.1 任务报告

## 任务描述

减少子任务容器的 margin-left 缩进：将基础值从 24px 降至 16px，层级增量从 16px 降至 12px。

## 修改文件

- `src/webview/components/TaskItem.tsx`

## 修改内容

第248行 `childrenStyle` 的 `marginLeft` 属性：

**修改前：**
```typescript
marginLeft: `${24 + calculateTaskDepth(task.id) * 16}px`,
```

**修改后：**
```typescript
marginLeft: `${16 + calculateTaskDepth(task.id) * 12}px`,
```

## 缩进效果

| 任务层级 | 修改前 (24+depth*16) | 修改后 (16+depth*12) | 减少 |
|---------|---------------------|---------------------|------|
| depth=0 (R1) | 24px | 16px | 8px |
| depth=1 (R1.1, R1.1.1) | 40px | 28px | 12px |

## 验证结果

- 基础 margin-left 从 24px 降至 16px
- 层级增量从 16px 降至 12px
- R1 和 R1.1/R1.1.1 的子容器缩进符合预期
