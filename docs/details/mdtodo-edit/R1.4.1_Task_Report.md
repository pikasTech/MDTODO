## 问题与发现

### 1. 依赖安装问题

**问题**：CLI 运行时提示 `Cannot find module 'commander'`

**解决**：需要手动安装 npm 依赖
```bash
npm install --prefix ~/.claude/skills/mdtodo-edit/scripts/
```

**建议**：将依赖写入 package.json 的 dependencies 而非 devDependencies，或提供安装脚本。

### 2. add 命令无 --title 选项

**问题**：SKILL.md 文档中 `add` 命令示例包含 `--title` 选项，但 CLI 不支持

**当前行为**：add 命令自动生成任务 ID 并添加模板内容，不接受手动输入标题

**建议**：如需支持 `--title` 选项，需要修改 CLI 源代码。
