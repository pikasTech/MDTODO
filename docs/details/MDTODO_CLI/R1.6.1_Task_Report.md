# R1.6.1 任务报告：更新命令实现

## 任务概述

执行 R1.6 任务：实现更新命令（update、complete、start），实现更新任务内容、标记完成、标记进行中的功能。

## 实现内容

### 1. 已实现的命令

#### complete 命令
- **功能**：标记任务为已完成
- **用法**：`mdtodo complete <id> [--file <path>]`
- **别名**：`done`
- **效果**：在任务标题后添加 `[completed]` 标记

#### start 命令
- **功能**：标记任务为进行中
- **用法**：`mdtodo start <id> [--file <path>]`
- **效果**：在任务标题后添加 `[in_progress]` 标记

#### update 命令
- **功能**：更新任务标题
- **用法**：`mdtodo update <id> <title> [--file <path>]`
- **效果**：替换任务标题行的内容

### 2. 核心实现

#### taskManager.ts 方法

**markComplete(taskId: string)**
```typescript
markComplete(taskId: string): TaskOperationResult {
  return this.updateTaskStatus(taskId, true, false);
}
```

**markInProgress(taskId: string)**
```typescript
markInProgress(taskId: string): TaskOperationResult {
  return this.updateTaskStatus(taskId, false, true);
}
```

**updateTaskTitle(taskId: string, newTitle: string)**
- 精确匹配任务标题行（`## R1` 或 `### R2.1`）
- 保留原始标题级别（`##` 或 `###`）
- 只更新任务 ID 和标题部分

**updateTaskStatus(taskId, completed, processing)**
- 使用行号精确匹配，避免子任务误匹配
- 正确处理 `[completed]` 和 `[in_progress]` 状态标记的添加和移除

结果：`## R1 测试更新命令 [completed]`
