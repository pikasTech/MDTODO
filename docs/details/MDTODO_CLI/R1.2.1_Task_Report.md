### 2. 重构 CLI 入口文件

**文件**: `cli/src/cli.ts`

- **子命令**: 8 个子命令分别实现不同功能
- **全局选项**: `--file`, `--json`, `--version`, `--help`

### 3. 命令结构

| 命令 | 别名 | 描述 |
|------|------|------|
| `list` | - | 列出所有任务 |
| `get <id>` | - | 获取指定任务详情 |
| `add` | - | 添加新任务 |
| `add-sub <parentId>` | - | 添加子任务到指定任务 |
| `delete <id>` | `remove` | 删除任务 |
| `complete <id>` | `done` | 标记任务为已完成 |
| `start <id>` | - | 标记任务为进行中 |
| `stats` | - | 显示任务统计 |

### 4. 全局选项

- `-V, --version`: 输出版本号
- `-f, --file <path>`: 指定 TODO 文件路径
- `--json`: JSON 格式输出
- `-h, --help`: 显示帮助信息

## 解决的问题

1. **访问修饰符问题**: 将 `loadFile`, `resolveFilePath`, `printTasks`, `printTaskDetails` 方法从 `private`/`protected` 改为 `public`，以便在子命令的 action 回调中访问

2. **类型兼容性问题**: 修复 `fileService.ts` 中的 `ReadFileOptions` 和 `WriteFileOptions` 接口，使用正确的 `BufferEncoding` 类型
