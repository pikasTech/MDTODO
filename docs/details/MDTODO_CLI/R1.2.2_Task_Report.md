Usage: mdtodo [options] [command]

MDTODO CLI - 命令行任务管理工具

  -f, --file <path>   指定 TODO 文件路径
  --json              JSON 格式输出 (default: false)
  -h, --help          display help for command

Commands:
  list                列出所有任务
  get <id>            获取指定任务详情

  add-sub <parentId>  添加子任务到指定任务
  delete|remove <id>  删除任务
  complete|done <id>  标记任务为已完成
  start <id>          标记任务为进行中
  stats               显示任务统计
  help [command]      display help for command
