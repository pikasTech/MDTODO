### 1. list 命令

**文件**: `cli/src/cli.ts:112-129`

**功能**: 列出所有任务

```typescript
program
  .command('list')
  .description('列出所有任务')
  .action(async (options: CLIOptions, cmd: Command) => {
    const cli = new MDTODOCLI();
    const filePath = cli.resolveFilePath(cmd.parent?.opts().file);
    const { manager } = cli.loadFile(filePath);
    const tasks = manager.getTasks();

    if (cmd.parent?.opts().json) {
      console.log(JSON.stringify(tasks, null, 2));
      return;
    }

    cli.printTasks(tasks, 0);
    const stats = manager.getStats();
    console.log(`\n统计: 共 ${stats.total} 个任务，已完成 ${stats.completed}，进行中 ${stats.inProgress}，待处理 ${stats.pending}`);
  });
```

### 2. get 命令

**文件**: `cli/src/cli.ts:132-152`

**功能**: 按 ID 查询任务详情

```typescript
program
  .command('get <id>')
  .description('获取指定任务详情')
  .action(async (id: string, options: CLIOptions, cmd: Command) => {
    const cli = new MDTODOCLI();
    const filePath = cli.resolveFilePath(cmd.parent?.opts().file);
    const { manager } = cli.loadFile(filePath);
    const task = manager.getTask(id);

    if (!task) {
      console.error(`错误: 未找到任务: ${id}`);
      process.exit(1);
    }

    if (cmd.parent?.opts().json) {
      console.log(JSON.stringify(task, null, 2));
      return;
    }

    cli.printTaskDetails(task);
  });
```

### 3. JSON 格式输出

```bash

# get 命令 JSON 输出
```

### 4. 任务管理器核心方法

**文件**: `cli/src/core/managers/taskManager.ts`

```typescript
/**
 * Get all tasks
 */
getTasks(): TodoTask[] {
  return this.tasks;
}

/**
 * Get task by ID
 */
getTask(taskId: string): TodoTask | undefined {
  return this.findTask(this.tasks, taskId);
}
```

## 输出格式

### list 命令输出

- **树状格式**: 使用缩进显示任务层级关系
- **状态标记**: `[x]` 已完成, `[-]` 进行中, `[ ]` 待处理
- **统计信息**: 显示任务总数和各状态数量

### get 命令输出

- **任务ID**: 显示任务完整 ID
- **标题**: 显示任务标题
- **状态**: 显示任务当前状态
- **行号**: 显示在源文件中的行号
- **链接统计**: 显示链接存在情况 (存在数/总数)
- **子任务**: 如果有子任务也会显示

### JSON 输出

所有任务数据以结构化 JSON 格式输出，包含:
- id, title, description, rawContent
- completed, processing 状态
- children 子任务数组
- lineNumber, linkCount, linkExists
