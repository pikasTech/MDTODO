# VSCODE插件开发总需求说明

## 项目目标

开发一个 VSCODE 插件，功能是渲染类似 `doc\review\20260108_TODO.md` 的 md 文件为带子任务的 TODO 列表界面。

## 核心功能需求

| 功能 | 说明 |
|------|------|
| 树形任务展示 | 将Markdown格式的TODO文件渲染为树形列表 |
| 层级展示 | 支持多级任务嵌套，主任务和子任务清晰区分 |
| 展开折叠 | 每个任务项可点击展开/折叠子任务列表 |
| 状态标记 | 每个任务可标记完成状态 |
| 状态持久化 | 完成状态保存回原md文件 |
| 实时预览 | 监听文件变化，实时更新界面 |
| 任务编辑 | 点击任务进入编辑状态 |
| 任务执行 | 调用Claude Code执行任务 |

## Claude Code集成

- 运行方式：使用"start"命令开启新终端窗口
- 任务指令格式：`"execute {path_to_md_todo_file} 中的 RXX 任务"`

## 技术栈

- TypeScript
- VSCode Extension API
- Webpack/esbuild 构建

## 开发注意事项

必须使用 `npm.cmd` 而不是 `npm` 来执行 npm 命令，否则 windows 下会无效。
