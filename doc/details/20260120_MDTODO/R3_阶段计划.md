# VSCODE插件阶段计划总览

本文档汇总所有开发阶段的详细执行计划。

## 阶段一：项目初始化与基础框架

详见 [R3.1](./R3.1_阶段一计划.md)

**目标**：搭建开发环境，创建项目基础结构，配置构建工具

**主要任务**：
1. 初始化VSCODE插件项目
2. 配置构建工具（Webpack/esbuild）
3. 创建基础目录结构
4. 实现插件入口和激活逻辑

## 阶段二：核心TODO列表功能

详见 [R3.2](./R3.2_阶段二计划.md)

**目标**：实现TreeView界面，解析Markdown文件，显示任务列表

**主要任务**：
1. 实现Markdown解析器
2. 创建TreeView数据提供者
3. 实现任务树形展示
4. 添加展开折叠功能

## 阶段三：文件读写与状态同步

详见 [R3.3](./R3.3_阶段三计划.md)

**目标**：实现文件读取、状态保存、实时监听功能

**主要任务**：
1. 实现TODO文件读取
2. 实现状态保存（完成标记写入文件）
3. 添加文件变化监听
4. 实现实时预览同步

## 阶段四：Claude Code集成与任务执行

详见 [R3.4](./R3.4_阶段四计划.md)

**目标**：实现任务执行功能，调用外部Claude Code进程

**主要任务**：
1. 实现Claude Code调用接口
2. 创建终端执行进程
3. 实现参数传递
4. 实现执行状态反馈

## 阶段五：测试优化与功能完善

详见 [R3.5](./R3.5_阶段五计划.md)

**目标**：完善功能细节，优化用户体验，全面测试

**主要任务**：
1. 实现任务编辑功能
2. 优化UI交互细节
3. 添加错误处理
4. 完整功能测试
