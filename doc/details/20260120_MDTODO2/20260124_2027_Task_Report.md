## R53.7 删除 markdown 格式移除代码

### 执行操作

删除 `src/parser/index.ts` 中的以下 4 行代码：

```typescript
// 删除前
let title = content
  .replace(/\[Finished\]/g, '')  // 移除 [completed]
  .replace(/\[Processing\]/g, '')  // 移除 [Processing]
  .replace(/(R\d+(?:\.\d+)*)/, '')  // 移除任务ID
  .replace(/^##?\s*/, '')  // 移除开头的 ##
  .replace(/`([^`]+)`/g, '$1')  // 移除行内代码标记  <-- 删除
  .replace(/\*\*([^*]+)\*\*/g, '$1')  // 移除粗体  <-- 删除
  .replace(/\*([^*]+)\*/g, '$1')  // 移除斜体  <-- 删除
  .replace(/#{1,6}\s*/g, '')  // 移除所有等级的标题符号  <-- 删除
  .trim();

// 删除后
let title = content
  .replace(/\[Finished\]/g, '')  // 移除 [completed]
  .replace(/\[Processing\]/g, '')  // 移除 [Processing]
  .replace(/(R\d+(?:\.\d+)*)/, '')  // 移除任务ID
  .replace(/^##?\s*/, '')  // 移除开头的 ##
  .trim();
```

### 原因

这 4 行代码与第 256 行的注释矛盾：

```typescript
// 【修复R23.2】保留列表标记和其他markdown格式，在非编辑模式下使用marked渲染
```

移除这些代码后，marked 将能够正确渲染：
- 行内代码：` ``code`` `
- 跨行代码块
- 粗体文本
- 斜体文本
- 标题

### 验证结果

编译成功，现在可以在任务块中正常使用 markdown 代码格式。
