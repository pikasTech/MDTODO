# R51.11 任务报告

## 任务内容

标题栏的添加任务按钮也要改成和其他按钮类似的风格和高度，不要蓝色。

## 修改文件

- `src/webview/components/TaskList.css`

## 修改内容

将 `.add-task-header-btn` 样式从蓝色主题改为与其他 `toolbar-icon-btn` 按钮一致的深色主题：

**修改前：**
```css
.add-task-header-btn {
  background-color: #007acc !important;
  color: white !important;
}
.add-task-header-btn:hover {
  background-color: #0062a3 !important;
}
```

**修改后：**
```css
.add-task-header-btn {
  background-color: #1e1e1e;
  color: #888;
  border: 1px solid #3c3c3c;
}
.add-task-header-btn:hover {
  background-color: #3c3c3c;
  color: #d4d4d4;
}
```

## 效果

- 添加任务按钮现在与其他工具栏图标按钮（`.toolbar-icon-btn`）风格一致
- 移除了蓝色背景，使用深色背景 `#1e1e1e`
- 添加了边框 `#3c3c3c`
- 文字颜色从白色改为灰色 `#888`
- hover 时背景变为 `#3c3c3c`，文字变为 `#d4d4d4`
- 与刷新按钮、打开文件按钮等其他图标按钮保持统一风格
