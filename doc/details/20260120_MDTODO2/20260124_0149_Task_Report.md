# R51.10 任务报告

## 任务描述

折叠和展开按钮应该合二为一，变成一个带状态的按钮，且不同的状态有颜色区分。

## 实现方案

将任务列表中每个任务项左侧的展开/折叠图标改为带状态样式的按钮：

1. **折叠状态（可展开）**：
   - 背景色：浅灰色 `rgba(100, 100, 100, 0.15)`
   - 边框色：半透明灰色 `rgba(100, 100, 100, 0.3)`
   - 图标颜色：灰色 `#888`

2. **展开状态（已展开）**：
   - 背景色：浅绿色 `rgba(45, 122, 45, 0.2)`
   - 边框色：绿色 `rgba(45, 122, 45, 0.5)`
   - 图标颜色：亮绿色 `#4ade4a`

3. **悬停效果**：
   - 默认悬停：深灰色背景 `#3c3c3c`
   - 展开状态悬停：更绿的背景 `rgba(45, 122, 45, 0.3)`

## 修改文件

- `src/webview/components/TaskList.css` - 更新 `.expand-icon` 样式

## 效果说明

- 按钮尺寸：22px x 22px（略大于之前，更易点击）
- 圆角：4px
- 边框：1px 边框使按钮更突出
- 过渡动画：0.15s 平滑过渡

这样用户可以一眼看出哪些任务是展开的（绿色），哪些是折叠的（灰色），提升用户体验。
